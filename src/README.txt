Dans ce test:

Il y a une class qui contient le main pour lancer le programme: TesterSuperMarket.java

- Les stocks initiaux des articles sont initialisés dans le fichier SuperMarket.java


- Une fois que le programme est lancé, l'affichage et les saisit sont faite sur la console. Il faut choisir un produit définit de la manière suivante:

A => Apple; O =>Orange; W =>Watermelon 

Les caractères de choix sont A, O, W.

Si un autre caractère different est saisi on a un message d'erreur.

Choix de produit erroné A => Apple; O =>Orange; W =>Watermelon.

Ensuite une fois le produit choisit, on saisit le nombre de produit voulu, toute fois le nombre saisit doit être inférieur au stock disponible.

--> Pour chaque produit acheté, on affiche le prix selon la quantite saisi et le stock restant disponible.

Le programme s'arrête une fois que la quantité de l'article choisit au départ est épouisé.


-->Par exemple pour apple, si on choisit 2 : on affiche le prix que cela coûte et la quantité restante au stock.

-Le fichier SuperMarketTest.java est le fichier contenant les tests unitairs du programme.
